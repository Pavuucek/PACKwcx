PACK Plugin for Windows Commander by DarkOne
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Version 1.1 beta

How to install this plugin (32 bit only):

1. Unzip the WCX to the Wincmd directory (usually c:\wincmd)
2. In Windows Commander 4.0 (or newer), choose Configuration - Options
3. Open the 'Packer' page
4. Click 'Configure packer extension DLLs'
5. type  pak  as the extension
6. Click 'new type', and select the pack.wcx
7. Click OK

What it does:

It adds support of PAK file format
(used by games: Quake, Quake 2, Hexen 2 and others)
to Windows Commander. It allows you to extract, delete, add files
to/from existing & create new paks.
